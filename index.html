01 — Sobre mí

Aprendo construyendo

Soy estudiante de Desarrollo de Aplicaciones Web (DAW). Lo que más me define como desarrollador es que no me quedo en los ejercicios de clase: cuando veo un problema real, intento resolverlo con software.

El mejor ejemplo es StrAtisbo, mi proyecto principal: el registro estratigráfico en arqueología se sigue haciendo muchas veces en papel, así que construí una aplicación multiplataforma que lo digitaliza y genera la matriz de Harris automáticamente. Mantenerla me ha enseñado todo el ciclo de un producto: interfaz, base de datos, empaquetado para escritorio y móvil, releases y actualizaciones automáticas.

Me gustan las interfaces cuidadas, el software con un propósito claro y el código que se entiende dentro de seis meses.

Proyecto destacado

StrAtisbo

Aplicación de escritorio y Android para el registro estratigráfico en arqueología: fichas de unidades estratigráficas, relaciones entre capas y generación automática de la matriz de Harris. Funciona sin conexión, con base de datos local y actualizaciones automáticas.

Disponible para Windows, Linux, macOS y Android.
